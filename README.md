No-show prediction (Machine Learning)

This project predicts patient no-shows in healthcare using machine learning.

Dataset
Medical Appointment No Shows (Kaggle)

Methods
- Logistic Regression
- Random Forest
- XGBoost

Evaluation metrics
- ROC-AUC
- PR-AUC
- F1-score
- Recall

How to run
Open the Jupyter Notebook and run all cells.
